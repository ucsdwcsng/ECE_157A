# course

To begin with, make sure to run the install script

./setup.sh

========== <br />

To install the required out-of-tree (OOT) modules.  <br />

1.) go to gr-wes directory  <br />
2.) mkdir build  <br />
3.) cd build  <br />
4.) cmake ..  <br />
5.) make  <br />
6.) sudo make install  <br />
7.) sudo ldconfig  <br />
